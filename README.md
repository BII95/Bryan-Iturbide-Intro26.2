Bryan Iturbide's Portfolio
========================

Overview
--------

This project consists of a Personal portfolio website showcasing engineering projects, web development projects, and API integrations. The Open API project displays artwork and products from the Art Institute of Chicago.

Technologies Used
-----------------

*   HTML
    
*   CSS
    
*   JavaScript
    
*   GitHub API
    
*   Art Institute of Chicago API
    

Installation
------------

1.  Clone the repository:
```bash
git clone https://github.com/BII95/Bryan-Iturbide-Intro26.2.git
```
1.  Open the project folder.
    
2.  Open index.html in your browser.
    

Alternatively:

*   Open the project in VS Code
    
*   Install the Live Server extension
    
*   Right-click index.html
    
*   Select "Open with Live Server"
    

Using the Portfolio
-------------------

*   Browse the About, Experience, Skills, and Projects sections.
    
*   Submit messages using the contact form.
    
*   View GitHub repositories in the Projects section.

*   Connect via Social media options.
    

Using the Art Gallery
---------------------

*   Navigate to the Open API page.
    
*   Click "Next" to view random artwork.
    
*   Click "Random Product" to view museum products.
    
*   Use the navigation menu to jump between sections or back to portfolio.
